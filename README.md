🛟 Установка

⚠️ Устанавливайте в /root.

Ubuntu / Debian

sudo apt update && sudo apt install git python3 -y && \
git clone https://github.com/binary166/BinaryUserBot && \
cd Heroku && \
python3 -m venv .venv && \
source .venv/bin/activate && \
pip install -r requirements.txt && \
python3 main.py
